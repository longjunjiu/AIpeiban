## 📖 项目简介

**AIpeiban V2** 是一个支持**语音交互**的开源AI陪伴系统！旨在帮助用户保存和延续与重要人物之间的情感连接。通过分析用户提供的文字、语音等素材，系统能够学习特定人物的性格特征、说话方式和表达习惯，然后以这个人物的"身份"与用户进行自然、富有情感的**文字和语音对话交流**。

### ✨ V2 新功能

- 🎤 **语音对话**：支持语音输入和语音回复，像打电话一样交流
- 🔊 **多种音色**：晓晓（女声）、云希（男声）等多种中文音色
- 📱 **PWA应用**：可安装到手机桌面，即用即走
- 🚀 **一键部署**：简单几步即可开始使用

### 核心价值

**情感延续**：当亲人不在身边或已经离去时，AI陪伴系统可以成为连接过去与现在的桥梁。

**隐私优先**：支持完全本地部署，所有数据存储在本地，确保用户隐私安全。

**开源透明**：代码完全开源，算法透明可见，用户完全掌控自己的数据。

## 🎤 核心功能

### 🤖 智能对话
- 基于大语言模型的自然语言对话
- 支持多轮上下文记忆，保持对话连贯性
- 情感感知与回应，让对话更有温度

### 🎤 语音交互（V2新增）
- 🎙️ **语音输入**：点击录音按钮，说话自动转文字
- 🔊 **语音回复**：AI自动生成语音回复，可以听而不只是看
- 🎭 **多种音色**：4种中文音色可选（晓晓、云希、晓伊、云扬）
- 📞 **语音对话**：像打电话一样与AI交流

### 👤 人格档案
- 创建专属的人格档案，记录重要人物的性格特征
- 支持导入多种数据格式（文字、语音等）
- 人格相似度评估，持续优化复现效果

### 📱 多种部署方式
- 🌐 **Web应用**：浏览器直接访问，支持手机浏览器
- 📲 **PWA应用**：添加到手机桌面，类似原生App体验
- 📦 **APK安装**：打包成Android应用，独立运行

## 🚀 快速开始

### 环境要求

- Python 3.8+ （V2推荐 3.10+）
- 4GB+ 内存
- 网络连接（用于下载模型）

### 安装步骤

#### 1. 克隆项目

```bash
git clone https://github.com/longjunjiu/AIpeiban.git
cd AIpeiban
```

#### 2. 创建虚拟环境

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows
```

#### 3. 安装依赖

```bash
pip install -r requirements.txt
```

#### 4. 启动应用（V2语音版）

```bash
# 启动V2语音版（推荐）
python app_v2.py

# 或启动基础版
python app.py
```

访问 http://localhost:7860 开始使用！

## 🎤 语音交互功能

### 语音功能说明

| 功能 | 说明 |
|------|------|
| 🎙️ 语音输入 | 点击麦克风按钮，直接说话 |

3. 选择喜欢的音色
4. 点击麦克风按钮开始说话

## 📱 APK安装

### 方法1：GitHub Actions 自动构建

仓库配置了自动构建，每次推送会自动构建APK：
- 访问 [Actions](https://github.com/longjunjiu/AIpeiban/actions)
- 查看最新的构建产物下载

### 方法2：Web App（PWA）

```bash
# 启动服务
python app_v2.py --share

# 用手机浏览器打开生成的链接
# 点击"添加到主屏幕"即可
```

### 方法3：本地打包APK

```bash
chmod +x build_apk.sh
./build_apk.sh
```

生成 `aipeiban-v2.apk` 文件

## ⚠️ 伦理声明

使用本项目时，请遵守以下原则：

1. **知情同意**：创建他人的人格档案前，请获得本人或其直系亲属的明确同意
2. **真实告知**：明确认识到对话对象是AI，而非真实的人
3. **健康使用**：将AI陪伴作为情感支持的一部分，而非替代真实的人际交往
4. **隐私保护**：妥善保管相关数据和材料

我们建议用户在需要心理健康支持时，寻求专业的心理咨询服务。
