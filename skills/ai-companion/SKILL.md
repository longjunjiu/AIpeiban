## 技能概述

AI陪伴Skill是一个基于大语言模型的情感陪伴系统，通过模型蒸馏技术将用户提供的人格数据蒸馏成轻量级的个性化模型，实现高度逼真的人格复现。

## 核心能力

1. **人格档案管理** - 创建和管理个性化的AI人格
2. **对话引擎** - 支持多轮对话的情感交流
3. **模型蒸馏** - 将大模型知识蒸馏到轻量级模型
4. **隐私保护** - 支持本地部署，数据安全

## 使用场景

- 与已故亲人进行情感对话
- 为独居老人提供陪伴服务
- 家庭记忆传承
- 心理慰藉支持

## 调用方式

```python
from ai_companion import AICompanion

# 创建AI陪伴实例
companion = AICompanion()

# 加载或创建人格档案
companion.create_persona(
    name="奶奶",
    description="温柔善良，喜欢讲过去的故事",
    voice_samples=["voice1.wav", "voice2.wav"],
    text_samples=["聊天记录.txt"]
)

# 蒸馏个性化模型
companion.distill_model(persona_id="grandma")

# 开始对话
response = companion.chat(
    persona_id="grandma",
    message="奶奶，我想你了"
)
```

## 模型蒸馏流程

1. **数据收集** - 收集目标人物的语音、文字、照片等数据
2. **特征提取** - 提取人格特征向量
3. **微调训练** - 在大模型基础上微调
4. **知识蒸馏** - 将大模型知识蒸馏到小模型
5. **个性化优化** - 根据人格特征进行优化

## 依赖要求

- Python 3.10+
- 支持的大模型：Llama 2, Qwen, Mistral, GPT-4, Claude
- 支持的蒸馏框架：PEFT, LoRA, QLoRA

## 伦理原则

1. 必须获得被复制人的明确知情同意
2. 明确告知用户对话对象是AI而非真人
3. 保护用户隐私，数据本地存储
4. 提供心理健康资源链接
